# Dashboard

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.2.5.

## Run the project

* Clone the project 
* Navigate to sensat-test
* Run `npm install`

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.
